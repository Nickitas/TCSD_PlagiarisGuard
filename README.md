# DSTU_Anti-plagiarism
Web tool for checking texts for uniqueness in the electronic library of DSTU

## ğŸ—‚ About system :
The system is designed to detect plagiarism in text documents and help users create unique content. We use text comparison algorithms to determine the degree of similarity between documents, resulting in a report for user analysis.

The system has a simple and intuitive API testing interface, making it easy to use for all users. The code is written in Python and uses libraries for text processing and comparison algorithms. The front-end is written in React and uses Vite for fast development and build.
